# Magnet Ecommerce - Scaffold

## Frontend
- Tech: React + Vite
- Path: `frontend/`

### Run
1. `cd frontend`
2. `npm install`
3. `npm run dev`

## Backend
- Tech: Node.js + Express
- Path: `backend/`

### Run
1. `cd backend`
2. `npm install`
3. `Copy-Item .env.example .env`
4. `npm run dev`

## API Check
- Health endpoint: `GET http://localhost:5000/api/v1/health`
